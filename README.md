# Amterity - Modern Web Solution

![React](https://img.shields.io/badge/React-19.0.0-blue?logo=react)
![Vite](https://img.shields.io/badge/Vite-6.2.0-yellow?logo=vite)
![License](https://img.shields.io/badge/License-Private-red)

**Amterity** is a cutting-edge web application designed to deliver a premium user experience. Built with the latest web technologies, it features a dynamic and responsive interface suitable for modern business needs. This project showcases a comprehensive suite of pages including a landing page, services overview, team introduction, and more, all wrapped in a sleek design.

##  Key Features

*   **Dynamic Landing Page**: A high-impact entry point (`src/Page/LandingPage`) designed to convert visitors.
*   **Comprehensive Service Showcase**: Detailed sections (`src/Component/Services`) to explain offerings.
*   **interactive Team Section**: Meet the people behind the project (`src/Component/Team`).
*   **Awards & Recognition**: Highlight achievements (`src/Component/AwardPage`).
*   **Testimonials**: Social proof integration (`src/Component/Testimonials`).
*   **Pricing Plans**: Clear and attractive pricing tables (`src/Component/PricingPage`).
*   **Contact Integration**: Easy-to-use contact forms (`src/Component/ContactUs`).
*   **Modern Hero Section**: Engaging visual introductions (`src/Component/Hero`).
*   **Fully Responsive**: Optimized for all devices using modern CSS practices.
*   **Smooth Animations**: Integrated with **AOS (Animate On Scroll)** for a polished feel.

## Technology Stack

This project is built using a modern frontend stack:

*   **[React 19](https://react.dev/)**: The library for web and native user interfaces.
*   **[Vite](https://vitejs.dev/)**: Next Generation Frontend Tooling for blazing fast build times.
*   **[React Router DOM](https://reactrouter.com/)**: For seamless client-side routing.
*   **[React Icons](https://react-icons.github.io/react-icons/)**: Including popular icon sets like Font Awesome, Material Design, etc.
*   **[AOS](https://michalsnik.github.io/aos/)**: Animate On Scroll library to reveal elements on scroll.
*   **ESLint**: For maintaining code quality and consistency.
